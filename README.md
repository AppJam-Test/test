


### git 명령어

```
git branch -r : 원격저장소의 모든 브랜치 보이기

git checkout -t origin/develop

git push origin :id( branch 추가 )

git branch -r ( 원격제어중인 branch 띄우기 )

git push origin/id 

git checkout -t origin/develop

git push --set-upstream origin/브랜치이름 ( 브랜치 github에 올리기 )

git branch -D 브랜치 이름 ( 삭제 )
```
